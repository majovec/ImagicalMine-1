ImagicalMine is a third-party build of the popular Minecraft: PE server hosting software known as PocketMine. Utilizing ImagicalMine server owners can control almost every single aspect of the in-game experience. The team is currently re writing the software for an even smoother experience.

Our custom fork (currently being rewritten, but a stable branch is available) of PocketMine features heavily requested functionality not currently available in the official branch, and is the perfect solution for hosting large networks of servers.

Why ImagicalMine?

There is a multi-world system!

Create as many worlds as you want, and teleport between them freely. You'll never have to worry about getting bored with the current map. Switch it up.. Live a little! (Keep in mind, this requires a seperate plugin, but that plugin is provided in the extra files repository)

You have full control of your server!

Ban, teleport and whitelist your players. All done in one easy-to-use console. You can also manage all of your server's settings easily in the provided server.properties file.

You can have unlimited players!

Configure your router to allow port triggering and you're all set! Invite your friends, or even invite the entire world. Start your own community with ImagicalMine! You can easily configure this in the server.properties file! See this thread on the former forums [NOT IN USE FOR THIS SOFTWARE ANYMORE] for the reccomended specifications for a lag free server and unlimited players!

You can have cheat protection!

Automatically kick fly hackers, prevents griefing, prevents third-person PvPing, and tons of other anti-hack actions. Don't let a couple bad eggs deter your loyal players! This is a built in, exclusive feature!

Lots more!

With ImagicalMine there is no limit on creativity! Create any type of community you want, and rest assured that the entire process will be smooth and worthwhile. This can include building cities with a currency of minerals, or building a giant server network with huge numbers of players! Knock yourself out with creations!
